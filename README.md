# Majd Al-Asala Website

موقع شركة مجد الأصالة للمقاولات العامة والتبريد والتكييف.

## Deployment

This project is configured to deploy on CranL/Replit with the following settings:

- **Deployment Type**: Dynamic
- **Build Command**: `npm run build`
- **Start Command**: `npm start`
- **Port**: 5000

## Environment Variables

- `PORT=5000`
- `NODE_ENV=production`
